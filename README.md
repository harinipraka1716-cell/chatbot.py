 🤖 Smart Assistant Chatbot

A Rule-Based Chatbot developed using Python and Tkinter that interacts with users through predefined rules and responses. The chatbot can assist users with banking services, ticket booking, greetings, and general queries through a simple and interactive graphical user interface.

---

## 📌 Project Overview

This chatbot uses predefined rules and if-else conditions to understand user input and generate appropriate responses. It is designed to simulate basic customer support interactions without using Artificial Intelligence or Machine Learning models.

---

## ✨ Features

- Interactive GUI using Tkinter
- Greeting and Help Responses
- Bank Account Opening Information
- Balance Enquiry (Demo)
- Loan Information

## 🔄 Working Process

1. User enters a message through the GUI.
2. The chatbot receives the input.
3. Input is converted to lowercase.
4. Predefined rules are checked using if-else conditions.
5. Matching response is generated.
6. Response is displayed in the chat window.
